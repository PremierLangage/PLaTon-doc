# Mise en forme de l'énoncé

La clé `question` supporte les balises Markdown. Celles-ci permettent une mise en forme simple de l'énoncé.

~~~
text ==
*texte en italique*

**texte en gras**

`verbatim`

Liste non-ordonnée:

* Item
* Item
    * Item
    * Item
* Item

Liste ordonnée:

1. Item
2. Item
3. Item
==
~~~
